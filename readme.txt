1) Перед запуском в IDE необходимо установить библиотеки из req.txt файла
2) Для запуска игры необходимо запустить main.py файл

3) Передвижение персонажа происходит стрелками на клавиатуре
4) Метать копье для того, чтобы убить врага, можно в разные стороны при помощи клавиш WASD
5) Все настройки меню сохраняются в файле settings.json

Приятной игры)
